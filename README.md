**About Me**

I'm a full-stack developer based in Switzerland, focused on building clean, functional, and production-ready digital products. I work across the entire stack — from backend infrastructure to polished frontend interfaces — with a strong eye for detail and a bias toward shipping.

I build independently, move fast, and care deeply about the quality of what I put out.

<br/>

**What I Do**

- Design and develop full-stack web applications from concept to deployment
- Build backend APIs and services with Node.js, Express, and Python/Flask
- Craft performant, accessible frontend interfaces in HTML, CSS, and JavaScript
- Architect databases using SQLite, PostgreSQL, and other relational systems
- Create SaaS-style products with real authentication, dashboards, and admin panels
- Develop for German-speaking and international audiences
